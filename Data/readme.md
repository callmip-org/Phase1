### This folder contains sub-directories for the data for the Phase 1a and 1b experiments within the 1st Phase of CalLMIP.

It also contains the general (not site specific) forcing files needed for site-scale simulations and calibrations. Specifically, atmospheric CO2 forcing file for the historical/transient runs. 

Forcing files for each site can be obtained from modelevaluation.org. See the Phase 1 protocol for further details.

The flux observation files can be found in sub-directories in this Data directory. Note in Phase1a-test there is only one file for the DK-Sor test site.

Note: our original plan was to provide N deposition forcing data for historical/transient runs; however, upon further discussion with participating modeling groups we have decided to allow each model group to decide what N deposition (or fertilisation, for crop sites) data they will use (as well as whether N cycling is included or not). This information will be detailed by each modeling group in the 1st phase description paper.
