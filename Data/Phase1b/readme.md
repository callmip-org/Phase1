This folder contains the flux observation files for the 21 sites to be used for the full "multi-site" calibration in Phase 1b. 

The files include NEE, Qle and Qh variables and their uncertainties.

Please see Table 1 Section 2.3 in the Protocol for information on PFT type and percent cover, soil layer depths, and soil texture (percent sand, silt and clay) for each layer (where available).

Full information on the sources of PFT and soil information will be documented in the CalLMIP Phase 1 protocol paper.

Note: no flux observation files are provided for the spatial validation sites but the PFT and soil information for the validation sites is provided in Table 2 Section 2.3 in the Protocol.
