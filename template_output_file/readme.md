### This folder contains an example template output .nc file 

All participating modeling groups should use this file to verify whether their output file is:
a. filename is correct
b. contains all variables requested with correct names
c. variable units are correct
d. following keywords are specified correctly in the global attributes: Model, CalLMIP_Expt, CalLMIP_Phase, Calibration_stage, and Cal_Val 
e. time variable correctly specified
f. lat/lon included and correctly specified

For further information see Section 7 in the Protocol and the Phase 1b Checklist in the Protocol.
